# DPS Logger Simulator

Placeholder for future DPS sensor simulator tools.

This directory is reserved for a serial simulator that can emulate a DPS
pressure sensor for development and testing without physical hardware.

The simulator may provide:

- a virtual serial device (e.g. `/dev/ttySIM`)
- ASCII responses compatible with the DPS protocol
- configurable timing and noise characteristics

The simulator is not required for normal DPS Logger operation and is not
included in the core workflow of version 2.

This directory currently contains no implementation.
